Comment ajouter facilement Google Fonts à votre site WordPress

1 Feb 2013
Developpement Web

Avec l’arrivée de CSS3, il est bien plus facile d’avoir un site internet attractif. L’une des possibilités qui s’offre désormais aux concepteurs de site internet, c’est d’ajouter une police de caractère originale et ce facilement.

Pour cela on peut utiliser Google Web Fonts qui permet de choisir parmi plus de 616 (à l’heure de l’écriture de cet article) polices de caractère, fonts en anglais, pour votre design.

Choisir la police sur Google Web Fonts

La première étape consiste à sélectionner la police de caractère qui convient le mieux à votre site. Vous pouvez choisir une ou plusieurs polices selon le design souhaité pour votre site internet. Mon conseil c’est tout de même de ne pas sélectionner plus de deux polices. Vous pouvez par exemple en avoir une pour les titres et une pour le texte.

Rendez-vous sur Google Web Fonts pour voir l’ensemble des polices disponibles. Grace aux onglets du haut de la page vous pouvez sélectionner un aperçu pour différent type de contenu : mot, phrase, paragraphe.

Une fois la police choisie, cliquez sur Quick use. Si la police le permet, choisissez un ou plusieurs styles pour cette font. Gardez en mémoire que plus vous en choisissez, plus le temps de chargement de votre page sera long.

Utiliser la police de Google Font dans WordPress

Pour maintenant utiliser dans votre site internet vous devez suivre la procédure suivante. Cette méthode pour insérer Google Font fonctionne à la fois pour les sites sous WordPress et pour les autres.

Rendez vous dans le fichier qui contient la balise HTML <head>Classiquement elle se situe dans le fichier header.php. Insérez le code que vous avez récupéré avec votre font :

<link href='http://fonts.googleapis.com/css?family=Fruktur' rel='stylesheet' type='text/css'>

Avec cette étape vous avez demandé au navigateur internet de télécharger la police de caractère directement sur le site de Google. Vous devez maintenant associer cette police à la balise que vous voulez.

Dans cette exemple je veux que mes titres utilisent ma nouvelle police. Allez donc dans votre fichier CSS, classiquement style.css et insérez le code suivant (ou similaire selon votre cas) :

h1, h2, h3, h4, h5, h6 {
   font-family: 'Fruktur', serif;
}

Explication : Fruktur correspond à la police que j’ai chargé. Utilisez le nom de votre police. Sherif correspond à la police par défaut que le navigateur doit afficher s’il y a un problème pour afficher votre Google Font.

Fonctions avancées

Charger plusieurs polices en une seule ligne

Si vous voulez ajouter plusieurs polices en une seule fois, vous pouvez les regrouper et utiliser le séparateur | .

<link href='http://fonts.googleapis.com/css?family=Fruktur|Stalemate' rel='stylesheet' type='text/css'>

Ajouter un effet à la police

Google teste une nouvelle fonctionnalité qui pour le moment n’est pas supportée par tous les navigateurs internet. Vous pouvez consulter la liste de tous les effets et leur compatibilité ici.

Par exemple si je veux utiliser l’effet « feu » je peux charger la police comme précédemment mais en ajoutant le nom de l’effet dans l’url.

<link href='http://fonts.googleapis.com/css?family=Fruktur|Stalemate&effect=fire' rel='stylesheet' type='text/css'>

Je peux ensuite simple utiliser cet effet en le ciblant comme classe d’un 

<div>
par exemple. Pour cela il faut utiliser la classe : « font-effect » suivit du nom de l’effet.
Ce qui donne :

<div class="font-effect-fire">Je suis en feu<div>
