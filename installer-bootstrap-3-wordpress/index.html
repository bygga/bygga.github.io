Comment installer facilement Bootstrap 3 sous WordPress

12 Oct 2013
Développement Web

Pourquoi utiliser Bootstrap et WordPress

Bootstrap est un outil très pratique développé par Twitter et open-source. Il s’agit d’une collection de CSS et de javascript qui vous permet de créer un site internet moderne et complexe en seulement quelque lignes de code.

Vous n’avez pas à vous prendre la tête pour créer un bouton ou une fenêtre modale. Il vous suffit d’utiliser la bonne classe CSS dans votre code, et le tour est joué.

Bootstrap 3 s’intègre très facilement avec WordPress. Il vous suffit soit d’installer un plugin ou bien copier quelques lignes de code dans votre thème.

Installer Bootstrap 3 avec un plugin

L’un des intérêt de WordPress c’est qu’il est possible d’installer facilement de nouvelles fonctions sans avoir à toucher une seule ligne de code. Rendez-vous dans l’onglet « extensions » et « Ajouter » puis chercher pour le plugin suivant : « WordPress Twitter Bootstrap CSS ». Vous pouvez aussi le télécharger directement sur le site de WordPress.

Installer Bootstrap 3 manuellement

Installer Bootstrap 3 manuellement dans WordPress est très simple. Presque aussi simple que d’utiliser un plugin finalement. Ouvrez le fichier function.php de votre thème et copier le code suivant:

function bygga_bootstrap_scripts()
{
    if (!is_admin()) {
        wp_register_script('bootstrapjs', '//netdna.bootstrapcdn.com/bootstrap/3.0.0/js/bootstrap.min.js', array(), '3.0.0'); 
        wp_enqueue_script('bootstrapjs');
    }
}
add_action('init', 'bygga_bootstrap_scripts');
function bygga_bootstrap_styles()
{    
    wp_register_style('bootstrapcss', '//netdna.bootstrapcdn.com/bootstrap/3.0.0/css/bootstrap.min.css', array(), '3.0.0', 'all');
    wp_enqueue_style('bootstrapcss');
}
add_action('wp_enqueue_scripts', 'bygga_bootstrap_styles');

Cette syntaxe est la méthode officielle recommandée par WordPress pour ajouter du CSS ou Javascript.

Vous n’avez pas besoin d’ajouter aucun fichier à votre installation. Les fichiers sont hébergé sur un server CDN. Sans rentrer dans les détails, c’est très bien pour vous puisque cela devrait rendre le chargement de ces fichier rapide lorsqu’un visiteur visite votre site.

Une fois que vous avez copié ces lignes, vous pouvez modifier votre thème pour y ajouter le CSS ou Javascript qui vous plait.

Par exemple pour ajouter un bouton, vous pouvez utiliser le code html suivant:

<button type="button" class="btn btn-primary">Primary</button>
