3 Plugins pour ajouter facilement des tableaux dans WordPress

12 Feb 2013
Developpement Web

Créer un tableau : pas toujours facile

Faire des tableaux en HTML et en CSS c’est l’une des choses les plus chiantes que je connaisse. Que vous soyez un novice ou bien un webmaster pro, vous détestez forcement ajouter tables et tableaux à votre site web.

Si le tableau ne comporte que deux lignes et deux colonnes la tâche est pénible mais pas insurmontable. Au delà ca devient simplement un casse tête et une perte de temps.

Même WordPress qui simplifie la tâche de tout le monde en offrant une interface simple qui permet de créer du code HTML et CSS complexe n’arrive pas à offrir une bonne solution pour ces fichus tableaux.

Alors encore une fois, quand le squelette de WordPress arrive à ses limites il faut se tourner vers les plugins Facebook. Et heureusement il y a des programmeurs qui ont pensé à nous et nous offre des outils simples mais efficace pour créer des tableaux aux allures pro en moins de 2min.

Petit tour d’horizon des plugins WordPress pour créer des tableaux HTML.

Plugin WordPress pour tableaux

TablePress

TablePress est un nouveau plugin qui à été introduit fin 2012. Il remplace le plugin WP-Table Reloaded qui est maintenant obsolete.
Ce plugin WordPress pour créer et ajouter des tableaux dans votre site internet est gratuit. L’utiliser est un jeu d’enfant. L’interface est très simple. Vous choisissez la taille de votre table et vous pouvez ensuite remplir les éléments comme si vous utilisiez Excel.

Vous pouvez créer n’importe quel type de tableaux avec ce plugin. Voici quelques exemples :

  • Classement championnat (sport, jeux vidéo)
  • Tableau comparatif
  • Liste de contact, clients
  • etc…

Il suffit ensuite d’insérer un shortcut dans votre page/article et le tour est joué.

Exemple de tableau par TablePress

TablePress permet aussi d’importer des données, supporte le tri de colonne, la recherche, les images, etc…

Go Pricing

Go Pricing pour WordPress est un plugin qui permet de créer des tableaux à couper le souffle. Le design est bien plus joli que le plugin gratuit TablePress et il constitue un atout incontestable pour les sites internet qui cherchent à vendre un produit ou un service.

Je vous laisse voir le résultat par vous même.

Plugin Go Pricing

Rendez-vous sur leur site pour voir plus de démo du design que vous pouvez espérer en utilisant ce plugin. Les tableaux réagissent aussi à la souris et changent de couleur ou de taille par exemple.

Ces tableaux sont aussi dynamiques et responsive. Cela signifie qu’il vont s’adapter à la taille de l’écran du visiteur. Un internaute sur un Iphone pourra donc toujours apprécier vos tableaux sans avoir à dézoomer.

Pricing Tables

Pricing Tables est un autre plugin WordPress payant mais de bien meilleure qualité qu’un plugin gratuit. Tout comme Go Pricing, et comme son nom l’indique, l’accent est mis sur le coté business des tableaux.
Si vous voulez comparer plusieurs produits et services, ce plugin est fait pour vous.
L’ensemble du design peut être changé et adapté à vos goûts.

Conclusion

Avant pour créer un tableau en HTML i fallait écrire ca :

<table border="1">
<tr>
<th>Header 1</th>
<th>Header 2</th>
</tr>
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
</tr>
</table>

Et le résultat ressemblait à ca :

Maintenant avec ces plugins WordPress vous avez uniquement besoin d’ajouter un bout de code à votre contenu et le tour est joué !
