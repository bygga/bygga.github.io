Favicon - Toutes les size et type à utiliser pour n'oublier personne

15 Sep 2013
Developpement Web

Qu’est-ce qu’un Favicon

Un favicon correspond à l’icône qui s’affiche juste avant le titre de la page internet que vous visiter. Dans Chrome par exemple, le favicon se retrouve sur la gauche dans chaque onglet.

Cet image est très utilisée puisqu’elle permet à vos visiteurs de rapidement identifier votre site internet lorsque plusieurs onglets sont ouverts.

Mais un favicon peut être utilisé également par le navigateur internet à d’autres endroits. Il peut être affiché par exemple lorsque vous ajouter une page à vos favoris, ou bien sous iOS et Windows 8 lorsque vous créez un raccourci vers un site internet.

Comment ajouter un favicon à votre site internet

Classique Favicon – Obligatoire

Ce que j’appelle le classique favicon, c’est l’icône qui est utilisée et affiché par les navigateurs internet. C’est donc la plus courante et aussi la plus simple à installer. Pour ce faire rien de plus simple. Il suffit de créer une favicon, de nommer le fichier favicon.ico et de le placer directement au plus haut niveau de votre site. (Dans le dossier parent).

Il n’y a pas besoin d’ajouter du code HTML dans ce cas. Tous les navigateurs internet vont chercher le fichier favicon.ico dans le dossier de votre site.

Favicon pour iOS 2.0+ et Android 2.1+ – Optionnel

Il s’agit de l’icône qui s’affiche lorsque vous sauvegardez un site internet sur la page d’accueil de votre téléphone sous Android et iOS.

Pour permettre cette fonction, ajouter la ligne suivante dans le header de votre site, dans la balise <head>.

<link rel="apple-touch-icon-precomposed" href="path/to/favicon-152.png">

Explication:

  • « path/to/favicon-152.png » correspond au chemin vers lequel vous avez votre image. Par exemple image/favicon-152.png.
  • L’image en question doit être au format .png
  • L’image doit avoir une taille de 152px x 152px.

Sous WordPress vous pouvez ajouter le code suivant dans votre fichier function.php.

function mon_favicon() { ?>
<link rel="apple-touch-icon-precomposed" href="path/to/favicon-152.png">
<?php }
add_action('wp_head', 'mon_favicon');

Favicon pour Internet Explorer 10 Metro – Optionnel

Comme pour les favicon ci-dessus, si vous utilisez IE10 vous pouvez avoir une image associé aux tuiles que vous créer. Utilisez le code suivant, également à placer dans la balise <head>.

<meta name="msapplication-TileColor" content="#FFFFFF">
<meta name="msapplication-TileImage" content="/path/to/favicon-144.png">

Dans ce cas, l’icône doit avoir une taille de 144x144px et être au format .png.

Extra Favicon pour iOS – Optionnel

Selon les versions d’Apple iOS et de la résolution de l’écran (Retina ou non Retina), il existe différente taille de favicon que vous pouvez ajouter. En voici la liste.

<!-- For iPad with high-resolution Retina display running iOS ≥ 7: -->
<link rel="apple-touch-icon-precomposed" sizes="152x152" href="https://www.bygga.fr/path/to/favicon-152.png">

<!-- For iPad with high-resolution Retina display running iOS ≤ 6: -->
<link rel="apple-touch-icon-precomposed" sizes="144x144" href="https://www.bygga.fr/path/to/favicon-144.png">

<!-- For iPhone with high-resolution Retina display running iOS ≥ 7: -->
<link rel="apple-touch-icon-precomposed" sizes="120x120" href="https://www.bygga.fr/path/to/favicon-120.png">

<!-- For iPhone with high-resolution Retina display running iOS ≤ 6: -->
<link rel="apple-touch-icon-precomposed" sizes="114x114" href="https://www.bygga.fr/path/to/favicon-114.png">

<!-- For first- and second-generation iPad: -->
<link rel="apple-touch-icon-precomposed" sizes="72x72" href="https://www.bygga.fr/path/to/favicon-72.png">

<!-- For non-Retina iPhone, iPod Touch, and Android 2.1+ devices: -->
<link rel="apple-touch-icon-precomposed" href="https://www.bygga.fr/path/to/favicon-57.png">

Comment créer une favicon

Pour les images au format .png vous pouvez utiliser Photoshop, Gimp, voir même paint. Commencez un nouveau fichier et sélectionnez la taille en pixel qui correspond à l’icône que vous voulez. Sauvegardez et utilisez ensuite votre client FTP pour envoyer l’image chez votre hébergeur de site internet.

Pour créer un fichier .ico, vous pouvez commencer par créer un fichier .png et ensuite aller sur le site internet Faviconer.com pour convertir l’image PNG en ICO. Il est même possible de dessiner à la main l’icône directement.
