Comment traduire un thème WordPress en francais

24 Jul 2012
Développement Web

L’anglais reste la langue principale des développeurs

Bien que la communauté francophone sur WordPress soit importante, tous les thèmes ne sont pas forcement traduits en français.
Cela est notamment vrai pour les thèmes gratuits qui sont souvent disponible uniquement en anglais.
Certain thèmes professionnels sont en revanche déjà disponible en français ce qui va faciliter votre développement. Choisissez donc ces thèmes en priorité, surtout si l’anglais est loin d’être votre point fort.

Il existe comme souvent deux manières d’arriver à nos fins :

  • Modifier le code à la main
  • Utiliser un plugin

Le plus facile étant comme souvent d’utiliser un plugin, je vais vous montrer comment traduire votre thème WordPress en français de cette manière. Notre sauveur s’appelle donc : Codestyling Localization

Installez le plugin codestyling localization

  • Rendez vous dans l’onglet Extensions – Ajouter dans le tableau de bord de votre site sous WordPress.
  • Dans le champ de recherche tapez « codestyling localization » puis installer le plugin de ce nom. Activez ensuite l’extension.

Ajouter une langue

  • Pour lancer « Codestyling Localization » rendez vous dans Outils –Localization.
  • Choisissez ensuite le fichier que vous voulez traduire : Thèmes, extensions, etc…
    Dans notre cas, il s’agit du thème WordPress que vous souhaitez traduire en français.
  • Cliquez ensuite sur Ajouter une nouvelle langue

  • Dans le menu qui s’affiche, choisissez fr_FR (ou bien la variante Suisse ou Belge selon votre choix). Puis cliquez sur créer le .po

  • Sur la nouvelle ligne créée (celle qui correspond à votre langue) choisissez Rescanner puis Scanner maintenant dans la fenêtre qui s’affiche

Traduire votre Thème WordPress en Francais

  • Cliquez sur modifier sur la ligne qui correspond à votre langage pour commencer à traduire votre thème WordPress. Une liste des mots qui sont traduisibles sera affiché.
  • A coté de chaque mot ou phrase à traduire cliquer sur modifier.
  • Une fenêtre s’ouvre. La partie du dessus correspond à la phrase en anglais telle qu’elle apparaît dans votre thème. Ne la modifiez pas. Entrez plutôt la traduction en français dans la case du dessous puis cliquez sur Save & Next

  • Une fois toutes les phrases traduite en francais, il ne vous reste plus qu’a cliquer sur Générer le fichier .mo
